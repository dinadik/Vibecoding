Budget Tracker Prompt 

Task 01 
Goal: Creating a simple budget tracker
Context: Create a simple python script that asks the user for total monthly budget, asks for 3
expenses, subtracts expenses from total budget, displays remaining balance
Constraints:
Expenses, income and other currency values should be numbers
Currency should display in LKR

Task 02 
Goal: Modify the budget tracker script to warn low funds
Context: If the balance is less than 500 LKR print “Warning: Low Funds” You should use a
color to grab user’s attention
Constraints:
This should happen after the remaining balance is displayed

Task 03 
Goal: Modify the program so the user can use the program multiple times
Context: Users should be able to enter expenses multiple times and stop entering when they
type “done”.
