Grade Calculator Prompt

Task 1
Goal: generate a grade calculator using Python
Context:
The programme should first ask for a student’s name
Ask for 3 subject marks.
Calculate the average
Display Pass if the average ≥ 40, otherwise Fail
Constraints:
Student name should be longer than 1 character
Student's marks should each be in the range 0-100 inclusive
Print an error message if any of these criteria are not met

Task 2
Goal: Modify the grade calculator to assign grades
Context: We need to add a grading system to this grade calculator. Use these guidelines to
grade a student.
Students who have more than 75 average marks for the above 3 subjects should get an A.
Marks beween 60-74 should get a B, between 40-59 should get a C and if the student got
marks below 40 he should be marked as failed.
Constraints:
Everyone should get a grade

Task 3
Add continuous execution to the program:
After displaying the result, the program should ask the user whether they want
to continue or exit.
If the user types Exit, the program should stop.
Otherwise, the program should run again and allow another student's marks
to be entered.

Task 4
Goal: Modifying the grade calculator’s output
Context:
Generate ASCII text for “Grade Calculator by Group_
09”
Display it on the splash screen. (while the calculator starts)
Update the previous code accordingly to the given instructions.
Display the output as described under the constraints.
Constraints:
Display the splash screen with ASCII text for “Grade Calculator by Group_
09” for
around 5 seconds with a growing and shrinking animation. (while the calculator
starts)
Output should be displayed as follows:
------------------------------

Average: 72.5

Display the ASCII text and the grading output clearly separated.
